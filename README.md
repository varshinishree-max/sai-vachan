# Sai Vachan

**Sai Vachan** is a minimalist Google Chrome extension designed to deliver a randomly generated quotation of **Bhagawan Sri Sathya Sai Baba** through a single user interaction.  
Developed during His **100th Centenary year**, the project explores how values-based content can be meaningfully integrated into everyday digital environments.

With a single click, devotees receive a thoughtfully presented quotation, allowing Swamiâ€™s message to gently accompany them through work, study, or quiet moments online.

Built with love and gratitude, Sai Vachan reflects an intention to keep technology aligned with the values of **faith, simplicity, and inner stillness**. The extension prioritizes simplicity, performance efficiency, and respectful visual presentation, ensuring minimal cognitive and technical overhead for the user.

---

## Scope and Functionality
- Displays one randomly selected quotation per interaction  
- Lightweight popup interface with static asset optimization  
- No background processes or persistent storage  
- Fully client-side execution  

---

## Technology Stack
- **HTML5** for structure  
- **CSS3** for layout and typography  
- **Vanilla JavaScript** for logic and randomization  
- **Chrome Extensions API (Manifest V3)**  

---

## Design Considerations
- **User Experience**: Distraction-free, immediate access, zero learning curve  
- **Performance**: Minimal file size and execution footprint  
- **Ethical Design**: No tracking, analytics, or data collection  
- **Accessibility**: Legible typography and consistent layout across devices
